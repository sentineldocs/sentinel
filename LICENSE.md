## 🛠️ Common Errors & Fixes

### ❌ Error: "Injection Failed" or "Failed to Attach"

**Cause:** Roblox updated and the current Sentinel version isn't yet compatible, or you launched Sentinel before opening Roblox.

**Fix:**
1. Close both Roblox and Sentinel completely
2. Check for a Sentinel update via the auto-updater or [download the latest version](https://sentineldocs.github.io/sentinel/)
3. Open Roblox first, join a game, then launch Sentinel and inject

### ❌ Error: "Sentinel Executor Detected as Virus / Blocked by Antivirus"

**Cause:** Antivirus software flags executor files as suspicious due to the way they interact with process memory. This is a known false positive — **Sentinel Executor is not a virus**.

**Fix:**
1. Open Windows Security → Virus & Threat Protection → Manage Settings
2. Scroll to **Exclusions** and click *Add an exclusion*
3. Select the folder where Sentinel is installed
4. Re-download Sentinel from the [official source](https://sentineldocs.github.io/sentinel/) if it was already quarantined

---

### ❌ Error: "Script Execution Failed" or Scripts Not Running

**Cause:** The script may use outdated functions, or it was written for a different executor's API.

**Fix:**
1. Make sure you're using scripts from trusted sources or the built-in Script Hub
2. Check if the script explicitly requires a specific executor (e.g., Synapse X only)
3. Try wrapping the script in a `pcall()` to catch hidden errors:
```lua

    -- your script here
end)
```
